gobook by ngerakines

A port of the python logbook web application to Go.

Gobook is a personal logging web application.

Dependencies

* github.com/Philio/GoMySQL * github.com/garyburd/twister/web * github.com/garyburd/twister/server

Download

You can download this project in either zip or tar formats.
